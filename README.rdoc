== To Do List

To Do List is a web-application that includes project with lists end tasks. 

== Getting Started

  bundle install --without production
  bundle exec rake db:create
  bundle exec rake db:migrate
  bundle exec rake spec:models
  Set variables to send e-mail in development.rb or production.rb:DOMAIN,NAME,PASSWORD end ADDRESS
  Go to http://localhost:3000
  You'll see a login screen
